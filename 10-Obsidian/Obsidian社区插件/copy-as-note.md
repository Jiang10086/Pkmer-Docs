## 概述

### 插件名称
Copy Note Content

### 主要功能
将当前打开的 Markdown 笔记的原始文本复制到剪贴板。

### 适用场景
适用于需要快速复制笔记原始内容的场景，无论是在 Obsidian 移动端还是桌面端都可使用。

### 核心特色
- 支持移动端和桌面端，移动端可将复制命令添加到工具栏，桌面端在左侧侧边栏有图标，方便快捷操作。
- 仅复制 Markdown 笔记（`.md`）的原始文件内容，而非渲染后的预览内容。

### 使用建议
- **移动端**：在“设置→移动”中，于“工具栏”添加“Copy current note to clipboard”命令，点击工具栏按钮即可复制。
- **桌面端**：点击左侧侧边栏的图标，或在命令面板中运行“Copy current note to clipboard”命令来复制。
安装可通过社区插件搜索安装，也可手动从 GitHub 下载文件放置到指定目录后启用。
