## 概述

### Strategist Toolkit插件总结
1. **主要功能**：一键或通过命令面板生成完整的、颜色编码的商业模式画布（BMC），以Obsidian Canvas文件形式呈现。
2. **适用场景**：适用于商业规划、项目策划等场景，帮助用户快速构建商业模式画布。
3. **核心特色**：
    - 提供命令面板操作和侧边栏丝带按钮触发生成功能。
    - 有模态框提示输入业务/项目名称，用于文件名。
    - 画布创建在当前活动文件所在文件夹或库根目录。
    - 九张卡片按经典BMC网格排列，坐标不重叠，卡片有不同颜色。
    - 卡片标题有表情符号增强，还有提示问题，每张卡片有占位符引导编辑。
4. **使用建议**：手动安装时，先执行`npm install`和`npm run build`构建插件，再将`main.js`和`manifest.json`复制到指定目录，最后在Obsidian设置中启用。开发时可使用`npm run dev`进入监听模式，`npm run build`进行生产构建。
