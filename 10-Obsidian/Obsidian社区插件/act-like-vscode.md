## 概述

### Act Like VSCode 插件总结
1. **主要功能**：让 Obsidian 拥有类似 VS Code 的使用体验，包括 VS Code 风格的标签页（单击预览、双击固定）、文件浏览器着色（根据文件是否归档高亮显示）、标签徽章（在文件浏览器同步显示笔记内标签）和导航栏图标（克制展示文件/文件夹图标）。
2. **适用场景**：适合习惯 VS Code 操作风格，想在 Obsidian 中获得类似体验的用户，便于高效管理和查看笔记。
3. **核心特色**：功能设计简洁，导航栏图标克制展示，降低心智负担；文件着色和标签徽章能直观呈现文件状态和标签信息。
4. **使用建议**：在 Obsidian 桌面端 1.8.9 或更新版本使用。可通过“设置 → Act Like VSCode”配置导航栏图标开关和标签颜色。还可将 CSS 片段复制到 `.obsidian/snippets/` 并在“设置 → 外观 → CSS 片段”中启用来自定义风格，但需注意未测试与其他主题的兼容性。
