## 概述

### Accordion File View插件总结
1. **主要功能**：以手风琴式预览列表展示Markdown和文本文件，可列出所选文件夹或整个库中的匹配笔记，一次展开一个文件进行预览，能将Markdown文件渲染为格式化预览，TXT文件以纯文本显示，还添加了丝带图标和命令面板入口。
2. **适用场景**：适用于需要快速浏览Markdown和文本文件，在不打开文件的情况下查看内容的场景。
3. **核心特色**：手风琴式预览，一次只展开一个文件，便于集中查看；支持Markdown文件格式化预览和TXT文件纯文本显示。
4. **使用建议**：通过丝带图标或命令面板打开插件；可选择设置文件夹范围来缩小文件列表；使用“Filter”按文件名或路径过滤；点击展开项目预览，或使用编辑按钮在编辑器中打开文件。若要开发，按相应命令操作；发布时注意提交和上传指定文件。
