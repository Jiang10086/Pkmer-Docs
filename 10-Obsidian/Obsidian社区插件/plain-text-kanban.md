## 概述

### 主要功能
Plain Text Kanban 是一款 Obsidian 插件，能将简单的嵌套 Markdown 列表转换为交互式看板，支持拖放操作、图像显示、清单功能、存档、标签和泳道设置。

### 适用场景
适用于使用 Obsidian 进行项目管理、任务规划等场景，用户可通过 Markdown 文件创建和管理看板。

### 核心特色
- 保持文件为纯文本格式，在任何编辑器中都可读。
- 看板支持水平滚动，列宽固定，可添加、删除、存档列，还能重命名和重新排序。
- 卡片可添加、删除和内联编辑，支持 Markdown 渲染和交互式复选框。

### 使用建议
- 安装可通过 Obsidian 社区插件或手动下载文件完成。
- 打开看板时，在文件资源管理器中右键点击 Markdown 文件，选择“Open as kanban board”。
- 了解文件格式，用 `- # Title` 表示列，`\t- ## Title` 表示卡片，`#hashtags` 为标签。
