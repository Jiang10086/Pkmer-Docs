## 概述

### Agent File Pane插件总结
1. **主要功能**：隐藏指定文件名（如`AGENTS.md`、`CLAUDE.md`等）在Obsidian文件资源管理器中的显示，并将它们列在一个专用侧边栏中，该侧边栏会镜像文件夹树结构。
2. **适用场景**：当指令文件在多个文件夹层级使用，导致文件资源管理器杂乱，但又需要能方便访问这些文件时，此插件可有效解决问题。
3. **核心特色**：
    - 精确隐藏：通过运行时样式表隐藏精确匹配的文件名，不影响类似文件名。
    - 侧边栏文件夹树：右侧边栏视图展示文件实际文件夹层级，可快速定位。
    - 可配置文件名：能在设置中添加、删除或恢复默认的跟踪文件名。
    - 自动同步：文件创建、删除和重命名时，侧边栏自动重新渲染。
4. **使用建议**：可通过 ribbon 上的“bot”图标或命令面板打开侧边栏。使用 Cmd/Ctrl 点击文件可在新窗口打开。安装可通过社区插件浏览器或手动下载文件放置到指定目录并启用。
