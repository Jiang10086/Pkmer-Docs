## 概述

### Property Annotations 插件总结
1. **主要功能**：在每日笔记中为布尔属性显示内联、仅用于展示的注释，如显示“<3 days since last>”，且不修改笔记内容。
2. **适用场景**：适用于使用每日笔记记录信息，且有布尔属性记录需求的用户，可直观了解属性上次为 `true` 的时间。
3. **核心特色**：
    - 自动检测每日笔记文件夹和文件名日期格式。
    - 能找出每个布尔属性上次为 `true` 的最近日期。
    - 提供多种设置选项，如标签样式（详细或紧凑）、键过滤（全部或白名单），还有高级设置。
4. **使用建议**：若想使用该插件，可通过 `npm install` 安装依赖，`npm run dev` 进行开发，`npm run build` 进行生产构建；也可手动将 `manifest.json`、`main.js` 和 `styles.css` 复制到 `<Vault>/.obsidian/plugins/property-annotations/` 目录。使用时可根据自身需求调整标签样式、键过滤等设置。
