## 概述

### 主要功能
Codex Workflow 是一个用于从当前 Obsidian 库启动 Codex CLI 的插件。它添加了一个功能区图标和命令来启动 Codex CLI，默认打开 Windows 终端，在当前 Obsidian 库目录启动，并可向 Codex 传递可配置的启动提示。

### 适用场景
适用于使用 Obsidian 管理个人知识库，且希望借助 Codex CLI 进行知识管理的用户。

### 核心特色
- 提供功能区图标和命令，方便启动。
- 支持 `{{vaultPath}}` 和 `{{activeFilePath}}` 变量。
- 包含紧凑的右侧边栏启动面板。

### 使用建议
- 该插件为手动/本地插件，需从 GitHub 发布资产或仓库源安装。
- 确保 Obsidian 桌面版、Codex CLI 及终端应用可用。
- 可在插件设置中配置终端和 Codex 命令、功能区图标启动方式及启动提示。
- 插件会在 `.obsidian/plugins/codex-workflow/.tmp` 下写入临时启动文件。
