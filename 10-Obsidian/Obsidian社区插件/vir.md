# Obsidian 插件：Vir

> [!Note] 插件名片
> - 插件名称：Vir
> - 插件作者：Djordje Marković
> - 插件说明：在你的Obsidian侧边栏里有一个Claude Code的大语言模型（LLM）维基。工作时，它能从你提炼的会话知识中调出相关笔记。

## 概述

### 主要功能
- 状态栏：显示 `vir` 守护进程的实时健康状态（健康、陈旧、关闭、未找到 CLI），点击可打开设置。
- 侧边栏面板：“最近” 标签扫描保险库中的 `vir` 蒸馏笔记，“相关” 标签查询与正在编辑笔记相关的笔记。
- 命令：可通过命令面板执行 “搜索保险库”“显示相关笔记”“打开设置” 等命令。

### 适用场景
适用于使用 Claude Code 进行工作，需要在 Obsidian 中快速获取相关笔记知识的用户。

### 核心特色
将 `vir` 蒸馏的 Claude Code 会话知识引入 Obsidian 侧边栏，在写作时能自动推送相关笔记，避免多次搜索。

### 使用建议
先安装并配置 `vir` CLI，通过 `npm install -g @djolex999/vir-cli` 和 `vir init` 完成。插件安装可从社区插件市场（待审核）搜索 “Vir”，或手动下载文件复制到指定目录。在插件设置中，点击 “检测” 解析 `vir` 二进制路径，再点击 “测试连接”。
