## 概述

### Split Web Viewer Modifier 插件总结
1. **主要功能**：
    - **Obsidian 版**：可将外链在分屏的 Web Viewer 中打开，支持复用同一 pane 切换 URL，Shift+点击开新 tab，Cmd/Ctrl+点击反转默认打开目标。
    - **Chrome 版**：Shift+点击可复用 Chrome 原生分屏的副屏 tab，扩展图标颜色反映当前 tab 是否在分屏中。
2. **适用场景**：适用于 Obsidian 1.8+ 桌面端和 Chrome 140+ 浏览器，解决用户不想频繁切屏或开新窗口，希望链接在旁边打开的需求。
3. **核心特色**：两个子项目理念一致，都致力于实现链接“在旁边打开”，且代码均为纯 JS，无构建步骤，开发和更新方便。
4. **使用建议**：
    - Obsidian 版按对应 README 安装，使用相应快捷键操作。
    - Chrome 版需先在 `chrome://extensions` 开启开发者模式，加载 `chrome/` 目录，且要先手动右键 tab 选择“Open in split view”。
