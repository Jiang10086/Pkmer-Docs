## 概述

### Chancli Font Size Resizer插件总结
1. **主要功能**：可快速增大或减小Obsidian笔记编辑器的字体大小，标题按比例缩放，正文独立缩放。
2. **适用场景**：适用于在实时预览、阅读视图和源码模式下使用Obsidian的用户，能满足不同场景下对字体大小的需求。
3. **核心特色**：可通过状态栏、命令面板和设置选项卡三种方式调整字体大小；字体大小设置在重启后仍会保留。
4. **使用建议**：
    - 若想快速调整，可点击状态栏 `𝐀 16px` 旁的 `−` / `+` 按钮。
    - 若需精确控制，可在命令面板选择相应命令或在设置选项卡中调整步长、最小/最大边界，也可拖动实时滑块。安装时按步骤操作，先运行 `npm install && npm run build`，再复制相关文件到指定目录，最后在设置中启用插件。
