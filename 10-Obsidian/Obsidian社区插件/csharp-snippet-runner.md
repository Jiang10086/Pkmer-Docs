## 概述

### CSharp Snippet Runner插件总结
1. **主要功能**：为Obsidian中C#代码块添加运行按钮，使用自动安装的CSharpRepl运行时执行代码片段，并将输出直接显示在笔记中。支持多种C#代码块标签，可添加脚本参数，能保存代码片段的输出和参数，笔记渲染时恢复保存内容。
2. **适用场景**：适用于在Obsidian中编写和调试C#代码的场景，方便开发者在笔记中直接运行代码并查看结果。
3. **核心特色**：自动安装CSharpRepl运行时，支持多种C#代码块标签，自带运行按钮，操作便捷；保存输出和参数，方便后续查看和复用。
4. **使用建议**：若要开发该插件，可按`npm install`和`npm run dev`操作；若要构建，执行`npm run build`；手动安装时，将`main.js`、`manifest.json`和`styles.css`复制到`<Vault>/.obsidian/plugins/csharp-snippet-runner/`目录。
