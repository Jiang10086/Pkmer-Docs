## 概述

### Random Note Plus插件总结
1. **主要功能**：可随机打开笔记，支持排除特定文件夹，有 ribbon 图标和命令“Open Random Note”，还设有带文件夹排除设置的选项卡，若过滤后无符合条件的笔记会给出提示。
2. **适用场景**：适用于想随机浏览笔记，但又不想包含某些特定文件夹内笔记的场景。
3. **核心特色**：相比核心随机笔记插件，增加了文件夹排除功能，且运行完全在 Obsidian 内，不进行网络请求，仅使用文件路径进行选择，不读取笔记内容，也不存储或传输文件列表。
4. **使用建议**：手动安装时，先构建插件，再将相关文件复制到指定目录，最后在 Obsidian 中启用；开发时可使用 `npm run dev` 开启 watch 模式；发布时需更新版本信息并创建 GitHub 版本标签。该插件最低支持 Obsidian 0.16.0 版本，支持桌面和非桌面环境。
