## 概述

### 主要功能
- 从 Jira 问题创建 Markdown 笔记，并自动更新。
- 可按自定义模板获取 Jira 问题并保存为笔记。
- 一键更新单个或所有跟踪的问题。
- 能从编辑器中选中文本创建新的问题笔记。
- 在设置中管理 Jira 连接和跟踪的问题。
- 状态栏按钮可快速更新当前问题笔记。
- 支持忽略 TLS 证书错误。

### 适用场景
适用于使用 Obsidian 记录和管理 Jira 问题的场景，方便将 Jira 问题同步到笔记中。

### 核心特色
可自定义模板，能灵活保存 Jira 问题为笔记；操作便捷，可一键更新问题状态。

### 使用建议
1. 安装可手动从 GitHub 下载并放入指定目录。
2. 配置设置时，需在“设置→Jira 插件”中输入 Jira API URL、用户名、密码等信息，还可设置笔记保存路径和模板。
3. 创建问题笔记可选中问题键后通过命令创建。
4. 更新问题可点击状态栏按钮或使用命令。
