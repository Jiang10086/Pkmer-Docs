## 概述

### Web Task Dashboard插件总结
1. **主要功能**：将第三方服务的任务集成到Obsidian工作区，在当前环境中展示任务，实现轻松的任务管理，无需切换上下文。
2. **适用场景**：适用于使用Obsidian进行知识管理，同时希望整合第三方服务任务的用户，可减少在不同应用间切换的干扰。
3. **核心特色**：能直接在Obsidian中查看第三方服务的任务，避免了在不同工具间频繁切换，提高工作效率。目前支持Tracks服务。
4. **使用建议**：该插件暂不支持通过社区插件安装，需手动安装。步骤为：打开最新版本下载`main.js`和`manifest.json`文件，在Obsidian保险库中创建`.obsidian/plugins/web-task-dashboard`文件夹，将下载文件复制进去，在Obsidian社区插件中启用。若遇到问题或有功能需求，可查看项目现有问题或新建问题，也可考虑为项目做贡献。
