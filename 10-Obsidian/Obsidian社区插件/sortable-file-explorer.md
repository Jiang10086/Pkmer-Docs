## 概述

### Sortable File Explorer插件总结
1. **主要功能**：提供一个替代的文件资源管理器，支持文件和文件夹的拖放排序、多选、批量操作、内联重命名、显示或隐藏文件扩展名、外部文件导入等功能。
2. **适用场景**：适用于需要对Obsidian库中的文件和文件夹进行自定义排序、批量管理的用户，尤其是在大型或嵌套较深的库中，能更清晰地管理文件。
3. **核心特色**：
    - 与默认文件资源管理器分离，不影响原生资源管理器使用。
    - 视觉层级更清晰，便于区分。
    - 能记住自定义排序和文件夹折叠状态，且可跨设备同步。
    - 支持手动拖放排序、多选和批量操作。
    - 有智能拖放区域和准确高亮显示。
    - 键盘操作友好，兼容移动设备和上下文菜单。
4. **使用建议**：若需要对Obsidian库中的文件进行灵活管理，可开启该插件。使用拖放功能进行自定义排序，利用多选和批量操作提高管理效率，根据需求显示或隐藏文件扩展名。在移动设备上可查看桌面定义的排序顺序。
