## 概述

### 主要功能
Vault Chat Agent是用于Obsidian库的AI助手插件。支持与AI进行对话，可搜索库、检查笔记并准备编辑内容供审核。具备多种搜索范围，支持多种模型和端点，能对本地库进行索引，有相关笔记侧边栏，提供文本命令，支持保存提示、手动工作流、编辑提案审核等。

### 适用场景
适用于在Obsidian中进行笔记管理、内容编辑、总结提炼、项目任务管理等场景。

### 核心特色
- 支持多种搜索范围和模型，可灵活配置。
- 具备计划模式，可只读调查后再准备编辑。
- 有上下文提及和内联自动完成功能。
- 能实时重新索引库。

### 使用建议
先按 `npm install` 和 `npm run build` 构建，将 `dist/` 内容复制到 `<vault>/.obsidian/plugins/vault-chat-agent/` ，在Obsidian社区插件设置中启用，再在插件设置里配置OpenAI兼容的提供商，部分远程提供商需API密钥。
