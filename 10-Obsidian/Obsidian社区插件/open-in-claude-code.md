## 概述

### 插件名称
Open in Claude Code

### 主要功能
可从命令面板或侧边栏丝带图标，将当前笔记所在目录或库根目录在Claude Code中打开。

### 适用场景
适用于使用Obsidian且需要在Claude Code中处理笔记相关目录内容的用户，特别是在macOS系统上的用户。

### 核心特色
1. 提供丝带图标，点击可直接在Claude Code中打开活动笔记所在目录。
2. 命令面板有两个命令，可分别打开活动笔记父目录和库根目录。
3. 自动同步插件，每次启动前确保目标目录继承库根目录`.claude/settings.json`中的所有`enabledPlugins`，若目录没有该文件则自动创建。

### 使用建议
- 仅支持macOS系统，且需安装Claude Code CLI并确保其在`PATH`中。
- 可通过社区插件或手动方式安装。
- 开发时可按指定步骤克隆仓库、安装依赖并运行开发命令。
