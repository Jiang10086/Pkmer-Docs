## 概述

### 主要功能
Brunet 插件支持在 Obsidian 中查看、编辑和运行 `.bru` 格式的 HTTP 请求。可打开 `.bru` 和 Bruno `.yml` 请求文件，进行可视化编辑，发送 HTTP 请求，并通过侧边栏浏览整个 Obsidian 库。

### 适用场景
适合开发者或需要在笔记环境中测试 API 请求的用户，在记录笔记的同时能方便地进行 API 测试。

### 核心特色
用户无需离开笔记界面，就能完成 API 请求的编辑与测试，通过侧边栏可快速浏览和运行库中的所有请求。

### 使用建议
1. 安装：在 Obsidian 的“设置 - 社区插件”中，关闭安全模式，搜索“Brunet”进行安装并启用。
2. 快速开始：将 Bruno 集合放入库中，从文件资源管理器打开请求文件，点击“发送”运行请求，结果会显示在“控制台”标签页。
3. 可通过“环境”标签页或“设置 - Brunet”选择 Bruno 环境，使变量能从 `environments/*.bru` 文件中解析。
