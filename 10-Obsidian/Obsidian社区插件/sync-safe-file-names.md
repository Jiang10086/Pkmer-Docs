## 概述

### 插件名称
Sync-safe file names

### 主要功能
确保Obsidian文件能在所有设备间同步，将文件名中的不安全字符替换为连字符“-”，还可自动重命名新创建或改名的文件，且不会覆盖现有文件。

### 适用场景
适用于在不同设备间同步Obsidian文件，且文件名含特殊字符导致同步失败的场景。

### 核心特色
与其他需手动执行的文件名处理插件不同，该插件可自动运行，保证新创建文件的文件名安全。

### 使用建议
1. 按常规方式安装该插件。
2. 自动重命名默认开启，可在插件设置中修改。
3. 打开新文件、编辑内容后，运行“Insert report of all unsafe file names”命令查看不安全文件名报告。
4. 若对重命名方案满意，运行“Rename all files to be sync-safe”命令完成重命名。
