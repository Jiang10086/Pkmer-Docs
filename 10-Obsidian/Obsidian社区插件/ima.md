# Obsidian 插件：Ima Vault

> [!Note] 插件名片

## 概述

### Ima Vault 插件总结
1. **主要功能**：将腾讯 IMA 知识库和笔记无缝集成到 Obsidian，支持搜索、浏览、导入和阅读。具体包括按标题或内容搜索 IMA 笔记、浏览 IMA 笔记本目录、在 Obsidian 侧边栏渲染笔记、一键将 IMA 笔记以 .md 文件形式导入 Vault、多页面间的前后导航，还可通过命令面板操作。
2. **适用场景**：适用于需要在 Obsidian 中使用腾讯 IMA 知识库的用户，方便统一管理和查看知识内容。
3. **核心特色**：实现 IMA 与 Obsidian 的无缝集成，支持原生 Markdown 渲染，提供便捷的搜索和导入功能，还有导航栈方便页面切换。
4. **使用建议**：推荐从社区插件市场安装，安装后需获取 IMA 凭证（Client ID 和 API Key）并在插件设置中输入。使用时可点击左侧侧边栏的 🔍 图标打开 IMA 面板进行操作。
