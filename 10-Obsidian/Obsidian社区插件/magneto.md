## 概述

### 插件名称
Magneto

### 主要功能
可将Obsidian 库同步到 GitHub 仓库，支持全量同步、推送到 GitHub、从 GitHub 拉取文件，还能设置定时自动同步。

### 适用场景
适用于需要在不同设备（包括移动和桌面端）同步 Obsidian 笔记，且希望将笔记备份到 GitHub 的用户。

### 核心特色
- 无需本地安装 Git，通过 GitHub REST API 实现同步，支持移动设备。
- 支持私有仓库，可自动创建（在令牌允许的情况下）。
- 可配置排除同步的文件夹和文件，支持图片、PDF 等二进制文件。

### 使用建议
- 推荐从社区插件安装，操作更简便。
- 建议采用“私有仓库 + 细粒度个人访问令牌（PAT）”的设置方式，安全性更高。
