## 概述

### StepVox 插件总结
1. **主要功能**：这是一款用于 Obsidian 的语音控制 AI 助手，可通过语音查询、编辑和创建笔记。具备端到端语音流水线，支持 Vault 工具操作，能进行网络搜索，接入多种 LLM，有单次和持续对话两种模式，支持打断，还能自定义助手人格。
2. **适用场景**：适用于希望在 Obsidian 中实现免手动操作，通过语音高效处理笔记的用户，如双手忙碌时仍需处理笔记的场景。
3. **核心特色**：端到端流式语音交互，支持多 LLM 接入，有灵活的对话模式，可打断助手说话，还能自定义助手性格。
4. **使用建议**：需使用 Obsidian 桌面端且版本 ≥ 1.7.0，准备好 StepFun API key 及 LLM API key，授予 Obsidian 麦克风权限。联网搜索需额外准备 Tavily 或 Exa API key。社区审核期间推荐用 BRAT 安装，输入 `Cheng-Zi-Qing/stepvox` 并启用。
