## 概述

### 插件名称
File Metadata

### 主要功能
在侧边栏展示文件元数据、文本统计信息和文档大纲，包含文件信息（如名称、路径、日期、大小）、文本统计（如字数、段落数、阅读时间）、链接与代码统计、文档大纲、图片元数据等。

### 适用场景
适用于需要快速查看文件详细信息、统计数据和文档结构的场景，帮助用户更好地管理和了解笔记文件。

### 核心特色
- 不触碰前置元数据，安全可靠。
- 各部分可折叠，方便查看。
- 可点击复制数据，右键有更多操作。
- 支持多种日期显示格式。
- 可通过Style Settings插件自定义样式。
- 可在设置中灵活开关各个字段和部分。

### 使用建议
可点击左侧功能区的 **ⓘ** 图标或从命令面板运行“File Metadata: Open File Metadata panel”打开面板。面板位于右侧边栏，会随文件切换或编辑自动更新。安装可通过社区插件搜索安装，也可手动下载文件放入指定目录后启用。
