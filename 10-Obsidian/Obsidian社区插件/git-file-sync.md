## 概述

### Git File Sync插件总结
1. **主要功能**：该插件可实现Obsidian笔记与GitLab或GitHub仓库的无缝同步，支持对单个笔记进行选择性同步，具备推送、拉取、查看差异和解决冲突等操作，还支持批量处理特定文件夹。
2. **适用场景**：适用于共享项目、选择性备份以及跨平台工作流，能让用户灵活控制同步内容。
3. **核心特色**：与全库同步方案不同，它提供了对同步内容的精细控制，用户可按需选择同步的笔记。同时，拥有同步状态视图，能清晰展示文件情况；内置差异查看器，可在同步前对比本地和远程的更改。
4. **使用建议**：若只需同步部分笔记，可利用选择性同步功能；对于特定文件夹内的笔记，可使用批量操作提高效率。在同步前，利用差异查看器确认更改，避免冲突。
