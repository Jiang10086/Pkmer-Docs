## 概述

### 插件名称
Canvas Markdown Side Editor

### 主要功能
- 一键在右侧编辑窗格打开画布卡片关联的 Markdown 文件。
- 支持实时预览和自动保存，可灵活调整编辑面板的停靠位置（右、左、上、下或浮动）。
- 具备标题模式，可将卡片折叠为一级标题显示。

### 适用场景
适用于使用 Obsidian Canvas 进行可视化思考，同时需要编辑 Markdown 卡片内容的场景。

### 核心特色
解决了在 Canvas 内直接编辑 Markdown 卡片空间局促的问题，让用户在不离开整体视图的情况下，能舒适地使用完整编辑器进行写作。

### 使用建议
- 点击 Canvas 卡片即可开始编辑，切换卡片时会自动保存。
- 若需快速查看卡片标题，可在设置中开启标题模式，便于发现缺少标题的卡片。
