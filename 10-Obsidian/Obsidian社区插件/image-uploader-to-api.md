## 概述

### 主要功能
该插件支持在拖放或粘贴图片时将其上传至任意 API 端点，并以 Markdown 格式插入图片的公共 URL。支持多种图片格式，还可选支持 `.pdf` 文档。

### 适用场景
适用于需要将图片或 PDF 上传至自定义服务器、Imgur、Cloudflare Images 等图片托管 API 的场景。

### 核心特色
- 支持拖放和粘贴上传，可并行上传多张图片。
- 能与任意图片托管 API 配合使用，可配置 HTTP 头和响应解析。
- 上传时有 `![Uploading...]()` 占位符显示。

### 使用建议
使用前需在设置中配置 API 端点、文件字段名、图片 URL 路径、PDF 处理方式和 HTTP 头。对于 PDF 处理，可根据需求选择保存本地、始终上传或每次询问。对于自定义上传 API，按示例格式准确填写各项设置。
