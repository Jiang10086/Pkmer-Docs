## 概述

### Agentic Git Sync插件总结
1. **主要功能**：实现Obsidian库及子模块与GitHub的双向同步，借助AI处理git内部操作，包括冲突解决、错误诊断、起草提交信息，还能自动初始化空仓库。
2. **适用场景**：适用于不了解git的Obsidian用户，既满足个人笔记的私密同步，也支持团队协作。
3. **核心特色**：
    - AI代理：将git复杂操作交给AI，提供可视化冲突解决界面和推理过程。
    - 灵活模式：个人知识私密保存，团队知识存于子模块，采用简单分支模型。
    - 隐形同步：后台定时拉取和推送，配置文件可随仓库迁移，新设备克隆自动恢复配置。
4. **使用建议**：在Obsidian的社区插件中搜索“Agentic Git Sync”进行安装启用。使用时可放心将git操作交给AI处理，遇到冲突可参考AI建议进行决策。
