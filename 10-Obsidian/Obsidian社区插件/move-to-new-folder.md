## 概述

### Move to New Folder 插件总结
1. **主要功能**：可将文件或文件夹移动到新创建的文件夹，无需输入完整目标路径。添加了“Move file to new folder...”和“Move folder to new folder...”选项，能从可搜索选择器中选父文件夹，用 Obsidian API 创建新子文件夹并安全移动目标，移动文件时保留正常链接更新行为。
2. **适用场景**：适用于在 Obsidian 中需要整理文件和文件夹，将其移动到新创建目录的场景。
3. **核心特色**：操作流程快，无需完整输入目标路径；支持从文件资源管理器和命令面板操作；可设置默认父文件夹为当前笔记所在文件夹。
4. **使用建议**：在文件资源管理器中，右键选择对应移动选项，选父文件夹、输入新文件夹名并确认；也可在命令面板中运行移动命令。该插件支持桌面和移动平台，但 macOS 和 iOS 未完全验证。
