## 概述

### API Designer插件总结
1. **主要功能**：可在Obsidian笔记中直观地设计和记录API端点，能创建包含请求和响应体的端点，定义属性的名称和类型，还能即时添加或移除字段。
2. **适用场景**：适用于需要在笔记中设计和记录API的开发者或相关工作者，无需离开笔记就能完成API设计，提高工作效率。
3. **核心特色**：拥有简洁、响应式的布局，能适配工作空间；API设计结果直接以Markdown格式保存；支持自定义主题，可通过JSON设置颜色方案。
4. **使用建议**：先在Obsidian中安装并启用该插件，打开要设计API的笔记，点击工具栏按钮添加端点，然后直观地定义请求/响应字段。若想自定义主题，可在“设置→API Designer”中添加JSON主题。
