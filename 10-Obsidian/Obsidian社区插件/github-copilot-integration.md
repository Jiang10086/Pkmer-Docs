## 概述

### 插件名称
GitHub Copilot Integration

### 主要功能
该插件将 GitHub Copilot 集成到 Obsidian 中，支持与文档聊天、生成笔记、改写文本等。具备聊天界面，可实时流式响应、自动包含文档上下文；支持内联编辑，选中文本后输入指令，Copilot 可直接在文档中追加或替换内容；能将选中文本作为上下文发送到聊天侧边栏并追问；还有动作面板，可通过模糊搜索快速访问 AI 操作。此外，有 8 种内置操作，如专业编辑、续写、总结、校对等。

### 适用场景
适用于需要快速生成笔记、对文本进行编辑优化、从文档中提取关键信息等场景。

### 核心特色
实现了 AI 与 Obsidian 的深度集成，提供实时流式输出，有持久的对话历史，支持原生 Obsidian 主题。

### 使用建议
可利用动作面板（Cmd+P → “Action Palette”）快速调用功能；选中文本后使用内联编辑和发送到聊天功能，充分发挥 AI 辅助作用。
