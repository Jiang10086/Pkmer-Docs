## 概述

### 主要功能
Created Daily Note Sync 是一款 Obsidian 插件，能根据 Markdown 元数据中的 `created` 字段自动创建缺失的每日笔记。支持自动启动、监控文件变化、读取元数据、创建对应笔记，还提供命令面板和文件资源管理器右键菜单操作。

### 适用场景
适用于有根据笔记创建日期生成对应每日笔记需求的用户，方便整理笔记。

### 核心特色
- 自动运行，实时监控文件变化。
- 提供命令面板和右键菜单操作，方便快捷。
- 优先使用 Obsidian 内置每日笔记配置，配置缺失时使用插件自身设置。
- 插件内通知、命令、菜单标签和设置界面使用中文。

### 使用建议
- 自动模式：启用插件后，编辑或保存带有有效 `created` 字段的 Markdown 文件，插件会自动创建缺失的每日笔记。
- 全库扫描：通过命令面板运行全库扫描命令，扫描库内 Markdown 文件并创建对应每日笔记。
- 右键菜单：在文件资源管理器中右键操作，可进行全库扫描或处理当前文件。
