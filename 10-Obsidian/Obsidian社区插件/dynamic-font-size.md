## 概述

### 插件名称
Dynamic Font Size

### 主要功能
通过前置元数据属性，借助直观的滑块界面动态调整笔记内容的字体大小。

### 适用场景
适用于需要灵活调整笔记字体大小，以满足不同阅读需求的场景，如在不同设备、不同环境下阅读笔记。

### 核心特色
1. 基于属性设置字体大小，可在笔记前置元数据中使用 `font-size` 属性。
2. 实时更新，修改属性后字体大小立即改变。
3. 提供交互式滑块，方便调整字体大小。
4. 状态栏显示当前字体大小。
5. 可自定义字体大小的最小、最大和默认值。
6. 字体大小变化有平滑过渡动画。

### 使用建议
可通过两种方式调整字体大小：一是手动在笔记前置元数据中添加 `font-size` 属性；二是使用滑块界面，通过命令“Toggle font size slider”调出滑块，在视图头部调整，调整后前置元数据会自动更新。
