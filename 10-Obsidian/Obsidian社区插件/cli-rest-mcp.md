# Obsidian 插件：REST and MCP server

> [!Note] 插件名片
> - 插件名称：REST and MCP server
> - 插件作者：Sébastien Dubois
> - 插件说明：这个插件还没经过Obsidian官方人员的人工审核。它能将命令行界面（CLI）命令以RESTful API端点的形式呈现，并且提供一个MCP服务器，方便与人工智能工具集成。

> - 项目地址：[点我访问](https://github.com/dsebastien/obsidian-cli-rest)
> - 国内下载地址：[下载安装](https://pkmer.cn/products/plugin/pluginMarket/?cli-rest-mcp)
> - 自述文件：[Readme](https://ghproxy.net/https://raw.githubusercontent.com/dsebastien/obsidian-cli-rest/master/README.md)



## 概述

### 主要功能
将所有Obsidian CLI命令转化为本地HTTP API和MCP服务器，可通过脚本、工具和AI助手对Obsidian vault进行编程式控制，实现笔记的创建、读取、搜索和修改等操作。

### 适用场景
适用于需要自动化笔记流程、让AI助手与笔记库交互、将Obsidian集成到现有工作流的场景。

### 核心特色
- 支持自动化笔记操作，方便提高效率。
- 可连接多种AI工具，增强交互性。
- 能通过标准HTTP请求进行集成。
- 安全性高，默认仅本地访问，有API密钥认证和精细的命令控制。

### 使用建议
- 确保使用Obsidian桌面应用v1.4.0及以上版本，并在设置中启用Obsidian CLI。
- 推荐通过社区插件方式安装，若社区目录未列出，可手动安装或使用BRAT安装获取最新版本，但BRAT方式可能不稳定。
