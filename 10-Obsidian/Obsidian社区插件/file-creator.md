# Obsidian 插件：File Creator

> [!Note] 插件名片
> - 插件名称：File Creator
> - 插件作者：Erin Skidds
> - 插件说明：这个插件还没经过Obsidian官方人员的人工审核哈。它能在特定文件夹里创建文件，还能选择给文件名加上日期前缀或者后缀，能根据模板生成PDF文件，也能创建看板。另外，它还能把附件嵌入到已有文件里呢。

## 概述

###  File Creator插件总结
1. **主要功能**：可在指定文件夹快速创建多种类型文件，如Markdown、PDF、看板、基础文件、Excalidraw绘图等；支持从模板创建PDF、Markdown、基础文件等；能将新创建的文件直接嵌入现有笔记。
2. **适用场景**：适用于需要快速创建各类文件、使用模板生成文件以及在笔记中嵌入文件的场景，适合知识管理、项目规划等工作。
3. **核心特色**：提供多种文件类型创建选项；支持文件夹选择和忽略特定文件夹；可选择在文件名中添加日期；有丰富的模板功能。
4. **使用建议**：因插件使用AI编码，若不希望AI代码进入库，请不要使用。使用时可根据需求选择合适的文件类型和模板，合理配置文件夹和日期选项，以提高文件创建效率。
