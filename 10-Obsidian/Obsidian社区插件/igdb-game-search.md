## 概述

### 主要功能
IGDB Game Search 插件可通过游戏标题搜索 IGDB 数据库，利用 IGDB API 获取游戏元数据，并根据这些数据创建游戏笔记。

### 适用场景
适用于游戏爱好者记录游戏信息，游戏开发者收集资料，以及游戏相关内容创作者整理素材等场景。

### 核心特色
能便捷地从 IGDB 获取游戏信息，快速创建游戏笔记，减少手动录入信息的麻烦。

### 使用建议
- 安装需手动进行，在 Obsidian 的指定文件夹放入相关文件后重启并启用插件。
- 使用时点击丝带图标或执行“Create new game note”命令，输入游戏标题搜索，从结果中选择游戏即可生成笔记。
- 使用前需获取 Twitch Client ID 和 Client Secret，按指引在 Twitch 开发者门户完成注册和设置后复制粘贴到相应位置。
