## 概述

### 主要功能
为搜索结果中的标签添加 CSS 类，可通过自定义 CSS 对其进行样式修改；提供设置开关，能一键隐藏搜索结果中的标签。

### 适用场景
适用于在笔记文本后使用标签分类笔记标题的场景，可让搜索结果更简洁，提升阅读体验。

### 核心特色
- 可自定义 CSS 类，对搜索结果中的标签进行个性化样式设置。
- 能一键隐藏标签，使搜索结果更清晰。

### 使用建议
- 安装：在 Obsidian 的“设置 - 社区插件 - 浏览”中搜索“Style Tags in Search Results”进行安装和启用，也可在 Obsidian 社区网站查看。
- 配置：在设置对话框中可编辑使用的 CSS 类，还能切换自动隐藏标签功能。
