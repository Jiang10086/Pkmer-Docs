## 概述

### 插件名称
Enchanted Animations

### 主要功能
为Obsidian中的所有元素添加动画效果，包括自定义动画速度和外观、切换特定 UI 元素的动画、控制几乎所有动画，还有动画复选框等。

### 适用场景
适用于追求更流畅、响应式界面体验，希望为Obsidian增添生动效果的用户。

### 核心特色
- 可对整个Obsidian的动画速度和外观进行自定义设置。
- 能针对特定 UI 元素开关动画。
- 持续更新，不断添加新的 UI 动画。

### 使用建议
1. 先通过链接https://community.obsidian.md/plugins/enchanted-animations ，点击“Add to obsidian”安装，再在Obsidian设置中启用插件。
2. 启用后，可对动画速度、缓动效果以及是否对特定 UI 元素添加动画等进行配置。若喜欢该插件，可给作者点赞或提交拉取请求提供帮助。
