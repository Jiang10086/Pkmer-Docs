# Obsidian 插件：Git Folder Sync

> [!Note] 插件名片
> - 插件名称：Git Folder Sync
> - 插件作者：Xheldon
> - 插件说明：它可以让你仓库里的文件与指定GitHub仓库中的特定目录（而非整个仓库）进行同步，还能把你的图片发送到云存储服务。
> - 插件分类：['文件管理', '第三方工具集成', '备份与恢复', '效率工具', 'obsidian插件']
> - 项目地址：[点我访问](https://github.com/xheldon/git-folder-sync)
> - 国内下载地址：[下载安装](https://pkmer.cn/products/plugin/pluginMarket/?git-folder-sync)
> - 自述文件：[Readme](https://ghproxy.net/https://raw.githubusercontent.com/xheldon/git-folder-sync/master/README.md)



## 概述

### Git Folder Sync插件总结
1. **主要功能**：支持与GitHub仓库双向同步，可将笔记同步到GitHub或从GitHub拉取；支持单文件操作、批量同步整个Vault；具备可视化配置界面；支持热重载开发；可递归同步文件夹结构；支持中英文；有智能缓存功能；状态栏显示文件同步状态和最后修改时间；粘贴图片时自动上传到云存储。
2. **适用场景**：适用于需要将Obsidian Vault中的文件与指定GitHub仓库目录同步的用户，方便文件的备份和共享。
3. **核心特色**：双向同步、单文件与批量操作结合，可视化配置界面直观，支持热重载开发，具备智能缓存减少API调用，支持多语言。
4. **使用建议**：手动安装需下载文件解压到插件目录并重启Obsidian；开发安装需克隆仓库、安装依赖，可通过`npm run dev`开启热重载开发模式，`npm run build`构建生产版本。配置时需先在GitHub创建个人访问令牌并设置相应权限。
