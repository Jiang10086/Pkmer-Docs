## 概述

### 1. 插件主要功能
Files Name Detector 是一款 Obsidian 插件，可自动将文件名中的空格替换为连字符。能对新创建的含空格文件名进行重命名，也会在插件加载时扫描并重命名库中现有含空格的文件名，重命名时会给出提示。

### 2. 适用场景
适用于希望统一文件名格式、避免文件名中出现空格的 Obsidian 用户，可提升文件管理的规范性。

### 3. 核心特色
自动处理文件名，无需手动逐个修改，节省时间和精力；能同时处理新文件和已有文件，确保整个库的文件名格式一致。

### 4. 使用建议
可选择手动安装，从最新版本下载 `main.js` 和 `manifest.json`，复制到库的指定目录，再在 Obsidian 设置中启用；社区插件列表安装方式即将推出。开发时可使用 `npm i` 安装依赖，`npm run dev` 开启监听模式，`npm run build` 进行生产构建。
