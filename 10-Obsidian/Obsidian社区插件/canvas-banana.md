## 概述

### Canvas Banana 插件总结
1. **主要功能**：深度集成 Gemini、OpenAI 等 AI 模型，支持在白板和笔记中进行智能对话、文本创作与图像生成。包括上下文感知对话、AI 图像生成、canvas 节点生成、笔记 AI 助手、增强的 Canvas 编辑等功能。
2. **适用场景**：适用于在 Obsidian 中进行创意构思、内容创作、图像设计等场景，无论是在白板上梳理思路，还是在笔记中撰写文章，都能借助 AI 提升效率。
3. **核心特色**：具有“上下文感知”能力，能精准理解节点或选段背景；支持多任务并发和提示词预设；提供流式响应、多种纵横比和分辨率选择；支持多 API 提供商及自定义 Base URL。
4. **使用建议**：使用前了解第三方服务条款和 API 使用费用，按需选择合适的 API 提供商；可利用提示词预设功能提高创作效率；在不同场景下灵活运用各功能，如在白板中进行对话探讨，在笔记中使用侧边栏副驾驶和智能悬浮编辑。
