## 概述

### 插件名称
Cover Image

### 主要功能
自动将笔记属性 `cover` 中链接的图片添加到笔记正文顶部。

### 适用场景
适用于需要为笔记添加封面图，使笔记更加美观和具有辨识度的场景，如撰写文章、记录学习资料等。

### 核心特色
默认限制封面图的最大宽度和高度为 256 像素，可通过修改 CSS 变量 `--cover-image-max-width` 和 `--cover-image-max-height` 来控制。

### 使用建议
- 先在笔记的属性中设置 `cover` 并链接图片。
- 若对默认的图片大小不满意，可通过修改 CSS 变量来自定义封面图的最大宽高。
- 注意链接的图片路径要正确，避免出现图片无法显示的问题。
