## 概述

### 主要功能
Hover Tooltips 是一款为 Obsidian 设计的插件，可添加支持 Markdown 的交互式弹出提示。支持鼠标悬停查看描述，左键点击可固定提示框，且提示框内支持完整的 Markdown 格式。

### 适用场景
适用于在笔记中对特定词汇进行解释说明，或添加额外信息，方便用户在阅读笔记时获取更多相关内容。

### 核心特色
- 交互式提示：鼠标悬停即可查看描述。
- 点击固定：可通过左键点击固定提示框，便于持续查看。
- Markdown 支持：支持多种 Markdown 格式，如加粗、斜体、代码、链接等。
- 内外链接：提示框内的链接可点击。

### 使用建议
使用 `{/{word/description}/}` 语法添加提示内容。例如 `{/{Obsidian/Powerful note - taking system with **Markdown** support}/}`。还可在设置中为调用提示菜单分配热键。
