## 概述

### 主要功能
Open or Create File 插件可添加自定义命令，基于预定义模式创建或打开文件，支持从模板创建新文件、使用基于日期的动态文件路径、时间偏移功能，还能自定义命令名称，可导入导出自定义命令配置为 JSON 文件。

### 适用场景
适用于维护日常笔记、周计划等常用文档。

### 核心特色
自动化创建常用文件，减轻认知负担；支持动态路径和时间偏移，灵活生成文件；可自定义命令名称便于快速访问。

### 使用建议
在 Obsidian 的社区插件设置中找到该插件，点击设置齿轮图标，添加命令并配置相关信息。配置完成后，可在命令面板中输入命令名称执行。如创建每周购物清单、明日日常笔记等，可参考示例配置具体参数。
