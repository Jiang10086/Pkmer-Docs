## 概述

### Custom Translate 插件总结
1. **主要功能**：调用本地 LibreTranslate 服务进行文本翻译，支持单词本管理，可收藏、存储单词详细信息，支持导入/导出单词本，具备双语界面，可自定义快捷键。
2. **适用场景**：适合在 Obsidian 中进行文本翻译、学习和管理英语词汇的用户。
3. **核心特色**：依赖本地翻译服务，能存储丰富的单词信息，支持音标，可自由配置快捷键，有双语界面且可切换。
4. **使用建议**：使用前需确保 LibreTranslate 服务已部署运行，可通过 Docker 一键部署，部署后设置 API 地址。安装可从 Obsidian 社区或手动进行。翻译文本可选中文字后通过右键菜单、命令面板或快捷键操作；单词本管理可选中单词添加，在侧边栏打开管理界面。还可导入示例单词本快速体验，可在插件设置中切换界面语言和设置快捷键。
