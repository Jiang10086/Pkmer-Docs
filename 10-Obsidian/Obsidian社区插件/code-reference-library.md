## 概述

### 插件名称
Code Reference Library

### 主要功能
- 新建代码片段，可记录语言、框架、标签、描述和代码块。
- 新建设计模式文档，涵盖类别和示例。
- 支持按标签搜索代码片段和模式。
- 提供参考侧边栏，可按语言筛选浏览代码片段和模式。
- 可在设置中配置代码片段和模式的文件夹。

### 适用场景
适用于开发者在Obsidian中构建个人编码参考库，方便随时查阅代码片段、设计模式等信息。

### 核心特色
以纯Markdown形式存储，方便管理和查阅，通过标签和语言筛选实现快速查找。

### 使用建议
打开命令面板，搜索“Code Reference”即可访问所有命令，利用标签和语言筛选功能高效查找所需内容。
