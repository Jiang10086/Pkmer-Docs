# Obsidian 插件：Simple Folder Note

> [!Note] 插件名片

## 概述

### 主要功能
将与文件夹同名的 `.md` 文件视为“文件夹笔记”，隐藏文件资源管理器中重复显示的文件夹笔记文件，点击文件夹即可在当前标签页打开对应笔记。

### 适用场景
适用于使用 Obsidian 进行知识管理，希望通过文件夹笔记来组织和整理信息的用户。

### 核心特色
- 简洁轻量，仅实现基本的文件夹笔记功能。
- 支持两种文件夹笔记布局，优先识别文件夹内的同名笔记。

### 使用建议
- 安装可选择手动下载 `main.js` 和 `manifest.json` 到指定目录，或使用 BRAT 插件添加测试版插件。
- 创建文件夹笔记时，可根据个人习惯选择将笔记放在文件夹内或与文件夹同级。
- 若遇到问题，可参考插件的 GitHub 仓库获取帮助。
