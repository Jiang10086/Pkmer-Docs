## 概述

### 插件名称
Github Sync (Multi-Platform)

### 主要功能
利用 GitHub REST API，实现桌面（Windows、macOS、Linux）和移动（iOS、Android）设备间笔记的无缝实时同步。

### 适用场景
适用于需要在多平台间同步笔记，且希望数据存储在私人 GitHub 仓库的用户。

### 核心特色
1. 原生支持移动设备，无需 Git 二进制文件。
2. 实时自动同步，文件修改后智能触发，有 5 秒防抖。
3. 无服务器架构，数据直传私人 GitHub 仓库。
4. 内置基于哈希的冲突检测，减少同步冲突。
5. 支持处理最大 10MB 的二进制文件。
6. 提供基于网页的可视化面板，展示写作进度和同步统计。

### 使用建议
若你有多平台笔记同步需求，可放心使用此插件。它无需本地 Git 环境，使用方便。使用时注意控制文件大小，尽量不超过 10MB，以保证同步效率。
