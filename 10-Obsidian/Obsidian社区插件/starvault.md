## 概述

### 1. 主要功能
StarVault 是一款用于管理 GitHub 星标仓库的 Obsidian 插件。可一键将 GitHub 星标仓库同步到本地，支持通过仓库名称和描述进行全文搜索，能添加、编辑和管理仓库标签，可按编程语言过滤，还能按星标数、分叉数或最后更新时间排序，支持软删除仓库并可随时恢复。

### 2. 适用场景
适用于需要管理 GitHub 星标仓库、快速查找特定仓库的用户，在日常开发、学习过程中，方便对感兴趣的仓库进行整理和检索。

### 3. 核心特色
数据本地存储，使用 IndexedDB 保存仓库信息、自定义标签和软删除状态，不向外部服务器发送数据；具备智能搜索、多过滤和灵活排序功能，操作便捷。

### 4. 使用建议
先从 GitHub Releases 下载最新版本，将相关文件复制到 Obsidian 保险库的插件文件夹。在设置中输入 GitHub 个人访问令牌，点击“Sync GitHub Stars”开始同步。可通过侧边栏进行搜索、过滤和排序，在详情视图查看仓库信息、添加标签等。
