## 概述

### Feynman - AI research assistant 插件总结
1. **主要功能**：作为知识库的研究助手，借助本地 Docker 运行，利用 Anthropic API 密钥开展研究工作。
2. **适用场景**：适用于需要在 Obsidian 知识库中进行深度研究、文献综述、审计等工作的场景。
3. **核心特色**：提供多种内置工作流，如深度研究、文献综述等；运行过程中可实时查看进度，包括工具调用、中间推理和引用；操作需用户明确批准，保障安全；生成的成果以 Markdown 文件形式存于指定目录，可直接在 Obsidian 中跳转查看。
4. **使用建议**：安装前确保 Obsidian 版本 >= 1.5.0，安装并运行 Docker Desktop，获取 Anthropic API 密钥。安装配置好插件且 Docker 服务器运行后，可通过点击图标或命令面板打开聊天面板，选择工作流并提交表单。也可直接从命令面板调用工作流，适合以当前笔记为主题的情况。
