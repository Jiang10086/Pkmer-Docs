## 概述

### File Commander 插件总结
1. **主要功能**：支持批量选择、删除、移动文件，批量打标签、替换标签、重命名 frontmatter 元数据属性；可对图片重命名、转换为相对或最简路径；能查找失效图片链接、未引用图片、无标签笔记、孤立笔记和空文件，还可按文件夹和标签筛选文件。
2. **适用场景**：适用于需要对 Obsidian 库中的文件进行批量管理和清理的场景，如整理文件、清理无效资源等。
3. **核心特色**：功能丰富，涵盖多种批量操作；支持图片处理和各类文件查找；具备右键菜单快速操作和文件列表按路径排序功能。
4. **使用建议**：可通过 Obsidian 社区市场、GitHub Release 或手动安装。使用时点击左侧边栏文件图标或用命令面板打开批量文件管理器，勾选文件后利用工具栏按钮进行操作，注意删除文件不可撤销。
