## 概述

### 主要功能
借助 L10n.dev 的 AI 翻译 API，将 Obsidian 笔记翻译成 165 种以上语言，同时保留 Markdown 格式和结构。

### 适用场景
适用于需要将笔记翻译成不同语言的场景，无论是在桌面端还是移动端都能使用。

### 核心特色
- 采用先进 AI 进行上下文感知翻译。
- 支持动态语言搜索，能快速找到目标语言。
- 记住上次使用的语言，一键重复翻译。
- 有三种输出模式：创建新笔记、替换当前笔记、追加翻译内容。
- 可选择保留 YAML 前置元数据，仅翻译笔记正文。
- 每次翻译后显示已使用字符数和剩余额度。

### 使用建议
- 需注册免费的 [l10n.dev](https://l10n.dev) 账号获取 API 密钥，每月有 30,000 字符的免费额度。
- 通过社区插件安装该插件，安装后在设置中粘贴 API 密钥，选择输出行为和是否翻译前置元数据。
- 打开笔记后，可通过命令面板、侧边栏图标或右键菜单触发翻译。
