## 概述

### 主要功能
Neural Composer 是基于知识图谱的 Obsidian 聊天插件。它运行本地 LightRAG 服务器，构建笔记图谱，支持通过图谱连接与笔记进行对话，还具备自动管理服务器、结合图与向量搜索、同步保险库、可视化知识图谱、连接远程服务器、对接 MCP 工具等功能。

### 适用场景
适用于需要深入挖掘笔记间关系、进行知识探索和问答的场景，帮助用户更好地利用 Obsidian 中的笔记资源。

### 核心特色
- 基于知识图谱查找关系，而非简单关键词匹配。
- 自动管理 LightRAG 服务器，无需手动操作终端。
- 使用 Ollama 时数据 100% 本地处理，保障数据安全。

### 使用建议
使用前需安装 Python 3.10+，并通过 `pip install "lightrag - hku[api]"` 安装依赖，Obsidian 版本需 1.7.2 以上。设置好监控文件夹，以便笔记保存时自动重新索引。可利用知识图谱视图进行可视化探索。
