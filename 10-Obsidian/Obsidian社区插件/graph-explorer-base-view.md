# Obsidian 插件：Graph Explorer Base View

> [!Note] 插件名片

## 概述

### 主要功能
- 提供自定义 Bases 视图，将笔记渲染为交互式力导向图。
- 支持按 Base 查询展示匹配笔记，可对节点按多种属性（如探索状态、置信度等）进行颜色区分，按连接数或统一大小显示节点。
- 具备交互式图画布，有平滑过渡效果，支持键盘导航；还有侧面板。

### 适用场景
适用于逐个探索、跟踪和扩展知识库笔记的场景。

### 核心特色
- 由 Base 驱动，与 Bases 过滤器、标签等自然配合。
- 有探索/未探索跟踪，笔记前端事项存储“探索”标志。
- 支持成熟度跟踪，有“毕业笔记”概念。
- 视觉语言丰富，节点形状和多种元素编码不同信息。
- 每个使用该视图的 Base 有独立设置。

### 使用建议
- 可利用 Base 查询精准筛选展示笔记。
- 借助探索/未探索标志和成熟度跟踪管理笔记。
- 灵活运用图的交互功能，如拖动节点、键盘导航等，提升操作效率。
