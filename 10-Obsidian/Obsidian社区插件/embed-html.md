## 概述

### 1. 主要功能
Embed HTML 是一款 Obsidian 插件，支持在笔记中嵌入 HTML 文件，可嵌入多种扩展名（如 htm、html 等）的 HTML 页面。

### 2. 适用场景
适用于需要在 Obsidian 笔记里展示 HTML 内容的场景，如嵌入网页、可视化图表等。

### 3. 核心特色
- 支持按默认或自定义宽高嵌入 HTML 文件。
- 可滚动到 HTML 文件中指定 ID 的元素。
- 能提取 HTML 文件中指定 ID 的元素。

### 4. 使用建议
- 可下载 [demo vault](https://github.com/mnaoumov/obsidian-embed-html-demo-vault/) 查看使用示例。
- 插件可从 [官方社区插件库](https://community.obsidian.md/plugins/embed-html) 安装。若想安装最新测试版，需先安装并启用 [BRAT 插件](https://community.obsidian.md/plugins/obsidian42-brat) ，再点击 [Install via BRAT](https://intradeus.github.io/http-pro...) 进行安装。
