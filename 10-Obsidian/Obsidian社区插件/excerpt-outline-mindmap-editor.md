## 概述

### 插件名称
Excerpt-Outline-Mindmap-Editor

### 主要功能
将 Markdown 大纲以可视化思维导图形式呈现，支持通过简单的拖放操作重新排列 Markdown 项目符号。

### 适用场景
适用于文献摘录、研究笔记等场景，借助空间视图便于导航和组织引用及想法。

### 核心特色
- 思维导图节点即 Markdown 列表项，移动节点会改变其在文件中的位置和缩进。
- 无自定义数据库、专有思维导图文件和隐藏文档格式，笔记在 Obsidian 普通编辑器中可读，且与其他基于大纲的工具兼容。
- 每个节点可用复选框置灰。

### 使用建议
1. 创建一个空的或已有大纲的 Markdown 文件。
2. 打开文件为思维导图：可在编辑器三点菜单中选择“Open as mindmap”，或在文件资源管理器中右键文件选择“Open...” 。拖放节点时，注意目标节点的四个放置区域，移动节点时其整个子树会随之移动，操作结果会同步到 Markdown 大纲中。
