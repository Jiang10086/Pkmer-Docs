## 概述

### CF ImageBed 插件总结
1. **主要功能**：将图片上传到 CloudFlare ImgBed，支持本地图片上传、网络图片抓取上传，可批量替换当前文档图片链接。
2. **适用场景**：适用于在 Obsidian 中需要管理图片并上传到图床的场景，如笔记撰写、文档编辑等。
3. **核心特色**：
    - 多种上传方式，包括拖拽、粘贴、右键选择和命令面板批量上传。
    - 网络图片接管，可抓取外链图片上传到自己的图床。
    - 支持一键批量替换文档图片链接。
    - 灵活配置，有多种上传渠道、命名方式等。
    - 跨平台，支持桌面端和移动端，移动端优化良好。
4. **使用建议**：
    - 手动安装需下载相关文件并复制到指定目录，重启启用；开发安装需克隆仓库并按步骤操作。
    - 配置插件时，要设置 API URL 和认证码或 API Token 等必要参数，还可进行可选配置，如选择上传渠道、文件命名方式等。开启网络图片上传功能时，合理设置排除域名。
