## 概述

### 插件名称
Open in File Explorer

### 主要功能
当在Obsidian编辑器中粘贴本地Windows文件系统的文件路径时，会提示将其转换为可点击的链接。

### 适用场景
适用于需要在Obsidian中引用本地文件路径，且希望通过点击链接快速打开对应文件或文件夹的场景。

### 核心特色
能自动转换粘贴的本地文件路径为可点击的Markdown链接，方便在Obsidian内直接访问本地文件系统资源。

### 使用建议
在需要引用本地文件路径时，直接将路径复制粘贴到Obsidian编辑器中，根据弹出的提示完成链接转换。转换后的链接可直接点击，若链接指向应用程序快捷方式，点击即可打开对应应用，如粘贴VLC媒体播放器的快捷方式路径，点击链接就能打开VLC。
