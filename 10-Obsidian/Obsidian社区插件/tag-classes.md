## 概述

### 主要功能
Tag Classes 是一款 Obsidian 插件，可自动将笔记的标签作为 CSS 类添加到视图容器，无需在笔记前言中手动添加 `cssclasses` 就能基于标签对笔记进行样式设置。

### 适用场景
适用于使用标签组织笔记，且希望根据标签对笔记进行样式定制的用户。

### 核心特色
自动将现有标签转化为 CSS 类，让标签兼具组织和样式设置的双重功能，无需额外添加 `cssclasses` 属性。

### 使用建议
- **安装**：可从社区插件商店搜索“Tag Classes”安装启用，也可手动下载 `main.js` 和 `manifest.json` 文件，放入 `<your-vault>/.obsidian/plugins/tag-classes/` 文件夹，重新加载 Obsidian 后启用。
- **使用**：启用后标签会自动作为类应用，无需额外配置。在 `.obsidian/snippets/` 中创建 CSS 文件，如 `tag-styles.css`，即可根据标签对笔记进行样式设置。
