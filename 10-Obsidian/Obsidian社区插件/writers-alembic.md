## 概述

### 主要功能
Writers Alembic 是一款适用于 Obsidian 的 AI 写作工作流工具，用户可创建和分享写作提示，工作流以 Markdown 文件形式存于保险库，能对其编辑、分享，并可在任意笔记上运行。

### 适用场景
适合需要借助 AI 进行写作辅助的 Obsidian 用户，如文案撰写者、内容创作者等。

### 核心特色
- 工作流以 Markdown 文件存在，方便管理和分享。
- 支持多种 AI 后端，如 Claude CLI、Anthropic API 等。使用 Claude CLI 无需 API 密钥，只要安装并认证 CLI 工具即可。

### 使用建议
- 先安装插件，可手动安装或通过社区插件安装。
- 打开设置配置至少一个 AI 后端。
- 打开笔记，可选择部分文本，从命令面板运行“Writers Alembic: Run workflow”，选择工作流即可在笔记中看到结果。
