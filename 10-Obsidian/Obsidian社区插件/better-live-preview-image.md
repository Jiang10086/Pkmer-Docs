# Obsidian 插件：Better Live Preview Image

> [!Note] 插件名片
> - 插件名称：Better Live Preview Image

## 概述

### 主要功能
- 提供图像对齐功能，可通过图像上下文菜单、命令面板或默认热键将图像左、中、右对齐。
- 支持实时预览图像的 Markdown 编辑，点击图像可显示其 Markdown 链接，保持选中框，且在右下角调整大小时 Markdown 链接仍可见。

### 适用场景
适用于在 Obsidian 实时预览模式下进行图像编辑的场景，帮助用户更好地处理笔记中的图像。

### 核心特色
直接编辑图像 Markdown，保证笔记在无插件时仍具可移植性和可读性；通过图像 alt 标记存储对齐信息。

### 使用建议
- 可在实时预览或阅读视图中右键点击图像选择对齐方式，也可在命令面板中运行相关命令。
- 利用默认热键（如 `Ctrl/Cmd + Alt + Shift + 方向键`）快速对齐图像，可在 Obsidian 设置的热键页面修改。
- 在插件设置中启用“Click image to edit Markdown in Live Preview”以开启实时预览 Markdown 编辑功能。
