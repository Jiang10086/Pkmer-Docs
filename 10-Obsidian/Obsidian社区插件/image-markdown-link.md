# Obsidian 插件：Image Markdown Link

> [!Note] 插件名片
> - 插件名称：Image Markdown Link
> - 插件作者：ariayd
> - 插件说明：请使用标准的Markdown语法 ![]() 来粘贴图片，而不是WikiLink格式，比如![[]]、[[]] 或者 []() 。

## 概述

### 主要功能
自动将粘贴到Obsidian中的图片从Wiki链接格式转换为标准Markdown图片语法`![](filename.png)`。

### 适用场景
适用于需要将Obsidian中的图片链接转换为标准Markdown语法，以便在其他Markdown渲染器中正常显示的场景。

### 核心特色
- 支持Obsidian可能自动生成的所有链接格式转换。
- 能处理任意深度的附件文件夹（根目录、子文件夹、相对路径）。
- 支持多种图片格式，如`png`、`jpg`等。
- 不修改非图片的Wiki链接。

### 使用建议
- 推荐从社区插件浏览器安装，安装后无需配置。
- 若插件不工作，先禁用再重新启用即可。安装后，打开Markdown笔记，直接从剪贴板粘贴图片，Obsidian会自动保存转换后的链接。
