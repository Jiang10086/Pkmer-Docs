## 概述

### Git Sync插件总结
1. **主要功能**：借助用户的免费GitHub账户，实现Obsidian笔记库在所有设备间的同步。自动创建以笔记库命名的私有仓库，每次保存文件时自动提交并推送，打开Obsidian时设备自动拉取最新更改，还能检测冲突并提供并排界面解决。
2. **适用场景**：适用于多设备使用Obsidian的用户，可在桌面（Windows、macOS、Linux）和移动（iOS、Android）设备间同步笔记。
3. **核心特色**：跨所有设备运行，一个GitHub账户可管理多个笔记库；自动后台同步，打开Obsidian即更新；有冲突解决界面；零成本，使用免费GitHub仓库，数据不离开用户账户。
4. **使用建议**：需Obsidian 1.0.0及以上版本、免费GitHub账户和网络连接，离线编辑会在联网后同步。若从源码构建，需Node.js 18+和npm。连接GitHub账户时在浏览器中一键授权即可。
