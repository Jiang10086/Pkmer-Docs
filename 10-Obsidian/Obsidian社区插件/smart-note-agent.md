## 概述

### 主要功能
Smart Note Agent 是一款为 Obsidian 打造的 AI 助手插件，可读取并经用户批准后修改笔记。具备多轮对话、多种工具调用、编辑笔记、支持多模型提供商、三种使用模式、定时任务、自动压缩对话历史、多语言界面等功能。

### 适用场景
适用于笔记问答、研究、编辑修改、日常总结等场景。

### 核心特色
- 自主工具调用循环实现多轮对话。
- 丰富的笔记操作工具，如全文搜索、读取笔记等。
- 支持多种大语言模型提供商。
- 提供三种使用模式，满足不同需求。
- 自动压缩对话历史，避免超出模型上下文限制。

### 使用建议
推荐通过社区插件安装，在设置中禁用安全模式后搜索安装并启用。使用时可根据需求选择合适的模式，如仅进行问答研究选“Ask”模式；需修改笔记选“Edit”模式；进行日常总结选“Scheduled”模式。同时可根据不同提供商配置独立的 API 密钥等信息。
