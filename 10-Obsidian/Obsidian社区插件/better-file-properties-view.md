## 概述

### Better File Properties View插件总结
1. **主要功能**：增强Obsidian的文件属性视图，具体包括可使用可配置的属性名称显示缩略图，在文件属性激活时可选择隐藏元数据容器。
2. **适用场景**：适用于需要更清晰查看文件属性，且希望对文件属性展示进行个性化设置的Obsidian用户，比如需要突出文件缩略图，或想精简属性展示界面的场景。
3. **核心特色**：通过简单配置即可实现文件属性视图的增强，可灵活调整缩略图显示和元数据容器的隐藏，提升文件属性查看的便利性和个性化程度。
4. **使用建议**：安装并启用该插件后，可在Obsidian设置中访问插件配置，根据自身需求进行个性化设置。若要自行构建项目，可使用“npm i”和“npm run build”命令。
